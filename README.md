## Descripcion

API para consumo y aplicacion de logica funcional para E-commerce Global Market Shop

## Instalacion

1. Instalar las dependencias
```
pnpm install
```

2. Levantar la base de datos
```
docker-compose up -d
```



## Correr la aplicacion

```bash
# Desarrollo
$ pnpm run start

# Desarrollo con actualizacion automatica
$ pnpm run start:dev

# Produccion
$ pnpm run start:prod
```
